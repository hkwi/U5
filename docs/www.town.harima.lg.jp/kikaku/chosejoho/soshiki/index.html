組織・部署から探す(播磨町機構図)

議会事務局グループ

  • 庶務・議事チーム
    本会議、委員会、請願陳情、議会関係条例

企画グループ

  • 政策調整チーム
    総合計画、行政改革、まちづくり、重要事項
  • 秘書情報チーム
    秘書、電算情報、広報、国際交流、市町交流

危機管理グループ

  • 防災安全チーム
    危機管理、防犯、防災、消防、交通安全

総務グループ

  • 人事文書チーム
    人事、文書、条例、規則、選挙管理
  • 財政管財チーム
    財政、財産管理、契約、庁舎管理

税務グループ

  • 住民税チーム
    町民税・県民税、国民健康保険税
  • 固定資産税チーム
    固定資産税・都市計画税、軽自動車税
  • 納税チーム
    収納徴収、納税相談、滞納整理

保険年金グループ

  • 国保年金医療チーム
    国民健康保険(資格・給付）、国民年金、福祉医療、後期高齢者医療制度
  • 介護保険チーム
    介護保険

福祉グループ

  • 高齢障害福祉チーム
    高齢者福祉、障害者福祉、介護手当
  • 社会児童福祉チーム
    民生委員、生活保護、児童福祉、母子福祉、児童手当、町営住宅、幼児保育
  • 子育て支援センターチーム
    子育て支援

すこやか環境グループ

  • 健康増進チーム
    健康管理、住民健診、予防接種、母子保健
  • 生活環境チーム
    環境保全、公害防止、廃棄物処理、施設管理

住民グループ

  • 戸籍チーム
    住民基本台帳、戸籍、証明書コンビニ交付、マイナンバーカード
  • 地域振興チーム
    自治会、住居表示、農林漁業、商工業、消費生活

都市計画グループ

  • 計画チーム
    都市計画、建築確認、都市景観、開発行為
  • 用地チーム
    用地取得、官民境界、未登記処理

土木グループ

  • 土木整備チーム
    道路・河川・用排水路・港湾・公園整備
  • 土木管理チーム
    道路・漁港・公園管理、道路占用許可、緑化推進

下水道グループ

  • 下水道整備チーム
    下水道（雨水・汚水）整備、排水整備
  • 下水道管理チーム
    下水道管・ポンプ場の維持管理、流域下水道、受益者負担金、水洗化の普及

水道グループ

  • 水道整備チーム
    給水装置工事、漏水修理
  • 水道管理チーム
    取水、浄水、施設管理、上下水道料金、給水受付

会計グループ

  • 会計チーム
    出納・審査

教育総務グループ

  • 学事チーム
    人事、予算、文書、条例、施設管理、奨学、保健、学校安全

学校教育グループ

  • 教育推進チーム
    幼児教育、学校園教育、家庭教育、給食、教職員、生徒指導、学制、就学

生涯学習グループ

  • 生涯学習チーム
    社会教育・生涯学習、スポーツ振興、文化芸術振興、施設管理
  • 人権推進チーム
    人権教育、男女共同参画
  • 郷土資料館
    郷土資料の収集・管理、埋蔵文化財
